## Build

```bash
# Build
zig build
# Upload using stlink-tools
zig build flash
# Rmemove .zig-cache folder
zig build clean
```

## Options

```bash
# Use C implementation
zig build -DNO_ZIG
# Enable Float formatting
zig build -DNEWLIB_PRINTF_FLOAT
# Add Custom GCCPATH
zig build -DARM_GCC_PATH=<path>
```
